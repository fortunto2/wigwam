wigwam

Wigwam is a tool for easy dependency fetching and building on Linux.

I created wigwam to ease installing scientific software (Torch, Caffe, OpenCV, etc) that often depends on dozens of libraries. Think of it as a simplistic clone of Homebrew that stores the know-how of building from sources scientific software.

Features:

  • doesn't require root
  • very easy to turn on and off, doesn't lock you in
  • lets you run several environments side-by-side
  • debuggable and traceable
  • modular, extensible and concise codebase (one Python file)

Limitations:

  • doesn't support different versions of a library in one environment
  • doesn't support installation recipe versioning

Documentation, bug tracker, source code, TODO and pull requests are on GitHub.

Examples

  • install torch

  • install caffe

  • install opencv

wigwam install torch -DPATH_TO_NVCC="/usr/local/cuda-7.5/bin/nvcc" -DPATH_TO_CUDNN_SO="/usr/local/cudnnR3/lib64/libcudnn.so"

Installing

Prerequisites: check that your system has Python 2.7, bash, wget/curl.

Append the following alias to your ~/.bashrc file (and don't forget to reload it):

alias wigwam='python <($([ -z "$(which curl)" ] && echo "wget -qO -" || echo "curl -s") https://raw.githubusercontent.com/vadimkantorov/wigwam/master/wigwam.py) --repo https://github.com/vadimkantorov/wigwam/tree/master/wigs'

This way will always run you on the latest version of wigwam and package scripts. Yet it can be unstable at times, for a more robust installation method refer to the documentation.
